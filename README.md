# Luv2Shop-eCommerce-Fullstack-Web-Application

Luv2Shop is a Full Stack Web Application for selling household items and computer accessories, providing a seamless shopping experience

Pre-requisites
```
@angular/cli  =>   17.3.8
Java          =>   JDK 17
Spring Boot   =>   3.3.0
MySQL	
```
